🔑 Claude Code 계정 관리

🔄 계정 자동 배정 방식 (새 Agent 노드 생성 시)
💡 "활성"(★)은 수동 모드 전용입니다.
번갈아 모드에서는 ★ 표시와 무관하게 등록된 모든 계정이 차례대로 사용됩니다.
각 Agent 노드는 생성 시 배정받은 계정을 계속 씁니다 (사이드바 [계정] 드롭다운에서 확인/변경 가능).
📝 사용 방법:
1. 본인 PC (Windows cmd / Mac 터미널)에서 실행:
   claude setup-token
2. 자동으로 브라우저 창이 열림 → Anthropic 로그인/승인
3. 승인 후 보여지는 코드를 복사해서 cmd에 돌아와 붙여넣기
4. cmd에 "✓ Long-lived authentication token created" 메시지와 함께
   sk-ant-oat01-xxxxx... 토큰이 표시됨
5. 그 토큰 문자열을 복사해서 아래에 붙여넣기
