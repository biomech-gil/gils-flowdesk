# tmux 사용법 가이드 (한국어)

## 시작하기

WSL Ubuntu 터미널을 열고 아래 명령어를 사용합니다.

```bash
# tmux 시작
tmux

# 이름을 지정하여 세션 시작
tmux new -s 작업이름
```

## 핵심 단축키

> **프리픽스 키**: `Ctrl+a` (이후 명령키를 누름)

### 세션 관리
| 단축키 | 기능 |
|---------|------|
| `Ctrl+a` → `d` | 세션에서 분리 (백그라운드 유지) |
| `tmux ls` | 세션 목록 보기 |
| `tmux attach -t 이름` | 세션 다시 연결 |
| `tmux kill-session -t 이름` | 세션 종료 |

### 창(Window) 관리
| 단축키 | 기능 |
|---------|------|
| `Ctrl+a` → `c` | 새 창 만들기 |
| `Ctrl+a` → `1~9` | 해당 번호 창으로 이동 |
| `Ctrl+a` → `n` | 다음 창 |
| `Ctrl+a` → `p` | 이전 창 |
| `Ctrl+a` → `,` | 현재 창 이름 변경 |
| `Ctrl+a` → `&` | 현재 창 닫기 |

### 패널(Pane) 분할 - 동시 작업의 핵심!
| 단축키 | 기능 |
|---------|------|
| `Ctrl+a` → `|` | 세로로 분할 (좌우) |
| `Ctrl+a` → `-` | 가로로 분할 (상하) |
| `Alt+방향키` | 패널 간 이동 (프리픽스 불필요) |
| `Ctrl+a` → `Ctrl+방향키` | 패널 크기 조절 |
| `Ctrl+a` → `x` | 현재 패널 닫기 |
| `Ctrl+a` → `z` | 패널 전체화면 전환 (토글) |

### 기타
| 단축키 | 기능 |
|---------|------|
| `Ctrl+a` → `r` | 설정 파일 다시 불러오기 |
| `Ctrl+a` → `[` | 스크롤/복사 모드 (vi키, q로 종료) |
| 마우스 스크롤 | 위로 스크롤 (마우스 지원 활성화됨) |

## 자주 쓰는 시나리오

### 1. 동시에 여러 작업하기
```bash
tmux new -s 개발
# Ctrl+a | 로 화면 분할
# 왼쪽: 코드 편집, 오른쪽: 서버 실행
```

### 2. 작업 중단 후 이어하기
```bash
# 작업 중 Ctrl+a d 로 분리
# 나중에 돌아와서:
tmux attach -t 개발
```

### 3. 여러 서버 모니터링
```bash
tmux new -s 모니터링
# Ctrl+a | 로 분할, 또 Ctrl+a - 로 분할
# 각 패널에서 다른 로그 모니터링
```

## 설정 파일 위치
- WSL: `~/.tmux.conf`
- 이 프로젝트: `tmux.conf` (백업용)
